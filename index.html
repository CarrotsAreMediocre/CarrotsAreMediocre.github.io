Starbound packets

All numbers are big-endian!

Base packet

The structure of the base packet is as follows:

Name Type Description
ID Byte The packet type, described later.
Payload size Signed VLQ The total size in bytes of the rest of the packet.
Payload Variable (described later) The actual useful data.

The ID lies between 1 and 48. The following is a list of the packet IDs and the type of packet it represents.
